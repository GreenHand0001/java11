## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

- 故障信息管理

#### 用户

- 查看公告

- 租金查询
- 故障申报
